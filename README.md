# Rock, Paper, Scissors Game

A simple console-based Rock, Paper, Scissors game written in C.

## Description
This project is a beginner-friendly C game where the player plays against the computer. The program asks the user to choose rock, paper, or scissors, compares it with the computer's random choice, and displays the result.

## Features
- Player vs. computer gameplay
- Random computer choices
- Score-based result display
- Simple and easy-to-understand C code

## How to Run
1. Compile the program using a C compiler:
command:gcc main.c
Then press Enter
After this step check in folder whether a.exe was there or not.
2. Run the executable:
   Command:./a.exe
   Enjoy the game!!

## Notes
This project is great for learning basic C concepts such as:
- Input and output
- Conditional statements
- Random number generation
- Loops
